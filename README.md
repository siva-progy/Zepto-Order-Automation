# 🛒 Intelligent Grocery Order Automation Platform

A production-ready workflow automation system that automates grocery ordering on **Zepto** using **n8n**, **Python**, **Selenium**, and **FastAPI**.

---

## 🚀 Overview

This project automates the complete grocery ordering workflow by:

- Receiving a product request
- Searching products on Zepto
- Identifying the correct product from similar results
- Adding the product to the cart
- Returning execution status through a REST API

The automation significantly reduces manual effort while maintaining high product selection accuracy.
